
Installation Guide

1. Compile the Code

Open the terminal and in the directory containing the Makefile run:

make

This will build the module, object fils in the build folder and the executable file "program.exe" in the directory you are currently in.

2. Run Tests [Optional]

To compile, and execute the tests for this code, use:

make test


3. Run the Program

After the compilation, execute the program with:

./program.exe


4. Makefile [Optional]

To see what single steps can be done with the Makefile, use:

make help
